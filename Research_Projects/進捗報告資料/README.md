# 進捗報告資料（月次）

研究室の月次ミーティング向けに、PowerPoint・PDF・画像などを置くフォルダです。

- パス: `Research_Projects/進捗報告資料/YYYY/YYYY-MM/`
- 月フォルダ名は `2025-05` のように年4桁・ハイフン・月2桁（ゼロ埋め）にします。

## レイアウト

- `YYYY/YYYY-MM/` … その月の資料一式。直下に置いても、`picture/` のようなサブフォルダを切ってもよい。

各月フォルダ内:

- `README.md` … 発表メモ・月報へのリンク
- その他 … `.pptx`、`.pdf`、画像

## 関連

- 月報: `Daily_Notes/YYYY/YYYY-MM_MonthlyReview.md`
